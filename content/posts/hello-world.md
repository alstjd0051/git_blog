## 블로그를 시작합니다

안녕하세요! TylerSong의 개발 블로그에 오신 것을 환영합니다.

이 블로그는 **Next.js**와 **Tailwind CSS**로 만들어졌으며, **GitHub Pages**를 통해 배포됩니다.

### 이 블로그에서 다룰 내용

- **개발** - 웹 개발, 백엔드, 프론트엔드 관련 이야기
- **AI / ML** - 머신러닝, 딥러닝 학습 기록
- **TIL** - 오늘 배운 것들을 정리

### 기술 스택

```
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Content: Markdown
- Deploy: GitHub Pages
```

앞으로 많은 글들을 작성해 나갈 예정입니다. 감사합니다!
